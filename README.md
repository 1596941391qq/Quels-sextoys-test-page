# 温柔的私语 – Quels sextoys ou accessoires de plaisir un père peut-il offrir à sa fille

这是一个纯实验性文档站点，用于测试 PSEO（程序化搜索引擎优化）在法语区低竞争长尾词的表现。

主题：以“父亲为成年女儿选礼物”的温馨叙事，探讨亲密护理与自我关怀的礼物建议。  
所有内容以资深亲密关系咨询师 + 暖心作者的视角撰写，零商业意图、零产品链接、零推销，仅分享知识与情感支持。

## 站点地址
- 预览：https://quels-sextoys-test.pages.dev/ （Cloudflare Pages）
- 文档版：https://quels-sextoys-test.readthedocs.io/ （Read the Docs）

## 技术栈 & 目的
- **MkDocs + Material for MkDocs**：纯 Markdown 文档生成，零框架依赖，轻量美观。
- **Cloudflare Pages**：免费托管、自动 CI/CD、全球 CDN 加速。
- **Read the Docs**：备用文档托管，SEO 友好，高信任子域名。
- **实验目标**：验证在法语区低竞争长尾词（如“cadeau intime pour fille adulte”“bien-être intime papa fille”）下，是否能在 Google 首页出现。

## 站点结构
- `/index.md`：首页 – 温柔引言 + 核心关怀分享
- `/faq.md`：常见问题 – 温和解答父亲的困惑
- `/about.md`：关于我 – 咨询师人设介绍
